# REST Blog App

A RESTful Blog Application built with React (Vite + Tailwind) and Node.js (Express + MongoDB).

## Tech Stack
- **Frontend**: Vite + React + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB

## Project Structure
- `/client` → React app
- `/api` → Express API
- `/docs` → Documentation (API contract, DB schema, etc.)
